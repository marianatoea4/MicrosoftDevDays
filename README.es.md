Una plantilla de taller para construir un **sitio web con temática de arcade retro** con Astro que enfrenta a dos usuarios de GitHub en una batalla de contribuciones. Este es el **punto de partida** — construirás la app paso a paso usando GitHub Copilot.

## ⚡ Qué Construirás y Aprenderás

Al final del taller habrás publicado un sitio de arcade retro completamente funcional **y** practicado estos superpoderes de Copilot:

| | Habilidad | En la práctica |
|--|-----------|---------------|
| 🧠 | **Ingeniería de Contexto** | Enseña a Copilot sobre tu código con instrucciones, referencias y restricciones |
| 📐 | **Planificar Primero** | Diseña la arquitectura antes de escribir una sola línea de código |
| 🤖 | **Implementación Agéntica** | Deja que Copilot realice trabajo de múltiples pasos bajo tu supervisión |
| 🎨 | **Diseño Iterativo** | Transforma visuales y refina detalles de interacción de forma interactiva |
| ⚡ | **Flujos Paralelos** | Divide el trabajo entre agentes, sesiones o tareas delegadas |

---

## 🛤️ Elige Tu Camino

El taller soporta **dos caminos** — sigue el que coincida con tu flujo de trabajo preferido:

| | Camino VS Code | Camino CLI |
|--|---------------|-----------|
| **Herramienta** | VS Code + extensión Copilot | GitHub Copilot CLI (`copilot`) |
| **Características clave** | Chat, Plan Mode, Agent Mode, background agents, review loops | contexto `@file`, `/plan`, ediciones autónomas, `/fleet`, `/delegate`, `/review` |
| **Ideal para** | Flujos nativos del editor | Flujos centrados en la terminal |

## 🚀 Inicio Rápido

1. **Crea tu propio repositorio** haciendo clic en **[Usar esta plantilla](https://github.com/marianatoea4/MicrosoftDevDays/generate)** arriba, o haz un fork de este repositorio.
2. Elige tu camino:
   - **VS Code:** clona tu repositorio y ábrelo en VS Code.
   - **GitHub Copilot CLI:** clona localmente, instala `copilot`, y trabaja desde tu terminal.
3. Sigue la [guía del taller](workshop/es/00-overview.md).

## ✅ Requisitos Previos

### Compartidos

- GitHub Copilot (Pro, Business o Enterprise)
- Git
- Node.js

### Camino VS Code

- VS Code v1.107+
- Extensión GitHub Copilot conectada

### Camino CLI

- GitHub Copilot CLI (`copilot`)
- Node.js 22+ para instalar via `npm install -g @github/copilot`
- O Homebrew / WinGet para instalación nativa
