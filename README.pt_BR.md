Um template de workshop para construir um **site com tema de arcade retrô** com Astro que coloca dois usuários do GitHub frente a frente em uma batalha de contribuições. Este é o **ponto de partida** — você vai construir o app passo a passo usando o GitHub Copilot.

## ⚡ O que Você vai Construir & Aprender

Ao final do workshop, você terá publicado um site de arcade retrô totalmente funcional **e** praticado esses superpoderes do Copilot:

| | Habilidade | Na prática |
|--|------------|-----------|
| 🧠 | **Engenharia de Contexto** | Ensine o Copilot sobre seu código com instruções, referências e restrições |
| 📐 | **Planejar Primeiro** | Projete a arquitetura antes de escrever uma linha de código |
| 🤖 | **Implementação Agêntica** | Deixe o Copilot executar tarefas de múltiplos passos com sua supervisão |
| 🎨 | **Design Iterativo** | Transforme visuais e refine detalhes de interação interativamente |
| ⚡ | **Fluxos Paralelos** | Divida o trabalho entre agentes, sessões ou tarefas delegadas |

---

## 🛤️ Escolha sua Trilha

O workshop suporta **duas trilhas** — siga a que combina com seu fluxo de trabalho preferido:

| | Trilha VS Code | Trilha CLI |
|--|---------------|-----------|
| **Ferramenta** | VS Code + extensão Copilot | GitHub Copilot CLI (`copilot`) |
| **Recursos principais** | Chat, Plan Mode, Agent Mode, background agents, review loops | contexto `@file`, `/plan`, edições autônomas, `/fleet`, `/delegate`, `/review` |
| **Ideal para** | Fluxos nativos do editor | Fluxos focados no terminal |

## 🚀 Início Rápido

1. **Crie seu próprio repositório** clicando em **[Use este template](https://github.com/marianatoea4/MicrosoftDevDays/generate)** acima, ou faça um fork deste repositório.
2. Escolha sua trilha:
   - **VS Code:** clone seu repositório e abra no VS Code.
   - **GitHub Copilot CLI:** clone localmente, instale o `copilot`, e trabalhe pelo terminal.
3. Siga o [guia do workshop](workshop/pt_BR/00-overview.md).

## ✅ Pré-requisitos

### Compartilhado

- GitHub Copilot (Pro, Business ou Enterprise)
- Git
- Node.js

### Trilha VS Code

- VS Code v1.107+
- Extensão GitHub Copilot conectada

### Trilha CLI

- GitHub Copilot CLI (`copilot`)
- Node.js 22+ para instalar via `npm install -g @github/copilot`
- Ou Homebrew / WinGet para instalação nativa
